# 音频播放速度控制功能

## 功能说明

在分析页面 (Analysis.vue) 中添加了音频播放速度控制功能，方便用户在识别节拍时调整播放速度。

## 使用方法

### 1. 速度滑块
- 在控制面板中找到"播放速度"区域
- 拖动滑块可以在 0.25x 到 2.0x 之间调整播放速度
- 速度值会实时显示在标签上

### 2. 快捷按钮（动态高亮）
提供了5个常用速度的快捷按钮：
- **0.5x** - 半速播放，适合精确识别节拍
- **0.75x** - 四分之三速度
- **1x** - 正常速度（默认状态）
- **1.25x** - 1.25倍速
- **1.5x** - 1.5倍速

**智能高亮功能：**
- 点击某个速度按钮时，该按钮会变成蓝色高亮
- 使用滑块调整到非预设速度时，所有按钮恢复默认颜色
- 如果滑块恰好调整到某个预设速度（如1.0），对应按钮会自动高亮

### 3. 音调保持
播放速度改变时会自动保持音调（preservePitch），音频不会因为速度变化而变形。

## 应用场景

1. **节拍识别** - 将速度降低到 0.5x 或 0.75x，更容易听清节拍点
2. **精确定位** - 慢速播放可以帮助更精确地双击波形添加节拍标记
3. **快速浏览** - 提高速度到 1.25x 或 1.5x 快速浏览整首曲目

## 技术实现

- 使用 HTML5 Audio API 的 `playbackRate` 属性
- 通过 WaveForm 播放器的 `setPlaybackRate()` 方法控制
- 支持 0.25x 到 2.0x 的速度范围，步进 0.05
- 默认启用 `preservesPitch` 保持音调不变
- 动态按钮高亮：使用 `isSpeedActive()` 函数判断当前速度是否匹配预设值
- 响应式设计：`playbackSpeed` 为 ref 变量，实时更新UI状态

## UI 特性

- 现代化的渐变色滑块设计
- 实时显示当前播放速度
- 快捷按钮便于快速切换常用速度
- 响应式布局，适配不同屏幕尺寸
